Research · MSc Thesis · 2025 THESIS
AUTOMATED RESEARCH SYSTEM

An autonomous, domain-agnostic information-gathering pipeline that uses LLMs and advanced search strategies to dynamically scrape, map, and structurally verify community health services across Ontario — addressing coverage gaps in manual directories like Ontario 211.

Built on four sequential nodes: a Query Expansion Matrix that prevents LLM generative convergence, a Crawl4AI web scraper converting raw HTML into structured Markdown, an LLM extraction engine with strict alignment prompting, and a custom 5-layer deduplication system that routes ambiguous entity pairs to an LLM context judge for final resolution.
